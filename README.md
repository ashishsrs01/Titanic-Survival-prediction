# Titanic-Survival-prediction

This project builds a machine learning model to predict passenger survival on the Titanic using the famous Kaggle Titanic dataset. The goal is to analyze historical passenger data and train a classification model that can estimate whether a passenger survived the disaster.

The project follows a complete end-to-end machine learning pipeline, including data exploration, preprocessing, feature engineering, model training, and evaluation.

Project Workflow

Data Exploration (EDA)
Understanding relationships between survival and features such as gender, passenger class, age, and fare.
Data Preprocessing
Handling missing values, encoding categorical variables, and preparing the dataset for model training.
Feature Engineering
Creating meaningful features to improve prediction performance.
Model Training
Training classification models to predict survival probability.
Evaluation
Measuring model performance using accuracy and other evaluation metrics.
Kaggle Submission
Generating predictions for the test dataset and formatting them for Kaggle submission.

Technologies Used
Python
Pandas
NumPy
Scikit-learn
Matplotlib / Seaborn
Visual Studio Code

Dataset
The dataset is taken from the Kaggle Titanic: Machine Learning from Disaster competition, which contains passenger information such as age, class, gender, and fare.

Goal of the Project
The purpose of this project is to practice data analysis, feature engineering, and machine learning modeling, while demonstrating a structured ML workflow suitable for real-world predictive tasks.
